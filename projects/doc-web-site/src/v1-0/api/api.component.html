class TranslationConfig {
  translationUrl: string
  sections: Array<string>
  defaultLanguage: string
}

translationUrl

The property defines the URL where NgTranslation will download the translation files from. The property value is a template string that has to contain 2 named parameters surrounded by curly brackets:
language
The code of the current or the requested language.
section
The name of a partial translation file without the file extension.

sections

The names of the translation files without the file extension. In case of lazy loaded modules the path of the module has to precede the file name, separated by a colon.

defaultLanguage
